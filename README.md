# BASE64 Encoder Decoder Program 
## setup
--------
```
pip install -r requirements.txt
```
--------
## excute with python 
--------
```
python base64_decode_qt.py
```
## make a excutable file (.exe file)
--------
#### In Powershell
```
./make_excutable_file.ps1
```

#### In cmd
```
./make_excutable_file.bat
```

#### In bash shell
```
./make_excutable_file.sh
```
